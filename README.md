# Lumina Guest Hub

Lumina Guest Hub is a premium, high-fidelity Hotel TV Application designed to provide a cinematic and intuitive experience for hotel guests. Built with modern web technologies, it features a rich, responsive interface with deep personalization and AI-driven interactions.

## 🌟 Features

- **Cinematic Dashboard**: A beautiful, glassmorphic interface with real-time weather, clock, and guest information.
- **Smart Concierge**: AI-powered chat interface for booking services, ordering dining, and getting local recommendations.
- **Entertainment Hub**: Integrated movie and series catalog with rich metadata and immersive hero visuals.
- **Service Tracking**: Real-time status tracking for room service, spa appointments, and housekeeping.
- **Interactive Map**: A high-end interactive resort map with point-of-interest navigation.
- **Accessibility Modes**: Specialized UI modes for high contrast and vision assistance.
- **Multi-Language Support**: Seamlessly switch between English, French, Italian, Spanish, Arabic, and Russian.

## 🛠️ Technology Stack

- **Framework**: [TanStack Start](https://tanstack.com/start) (React + TanStack Router)
- **Styling**: Tailwind CSS 4.0
- **Icons**: Lucide React
- **Animations**: CSS Transitions & Custom Keyframes
- **Deployment**: Optimized for Vercel and Cloudflare Pages

## 🚀 Getting Started

### Prerequisites

- Node.js (v18+)
- npm or pnpm

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/lumina-guest-hub.git
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

### Building for Production

To create a production build:

```bash
npm run build
```

## 📱 Vercel Deployment

This project is ready for one-click deployment on Vercel.

- Build Command: `npm run build`
- Output Directory: `.output` (Standard for TanStack Start)
